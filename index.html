Robotic Engineer

Perkenalkan, nama saya Gilang. Saya bekerja sebagai seorang Robotic Engineer, dengan fokus utama pada pengembangan teknologi robotik yang inovatif dan berkelanjutan. Dalam profesi ini, saya terlibat dalam desain, pembuatan, dan pengujian berbagai jenis robot yang dirancang untuk mempermudah kehidupan manusia, baik di sektor industri, medis, maupun kebutuhan sehari-hari. Dengan latar belakang pendidikan di bidang teknik dan hasrat besar untuk mengeksplorasi kecanggihan teknologi, saya berkomitmen untuk terus belajar dan berkontribusi dalam kemajuan dunia robotika.

Skills

Electrical Wiring, Electrical Design, PCB Design, Python programming language, C++ programming language Arduino IDE, Microcontrollers, Arduino.
